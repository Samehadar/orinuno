# Disclaimer

## Educational purpose

Orinuno is an open-source research and educational project that demonstrates
techniques for working with public video-catalog APIs, content metadata
aggregation, asynchronous job orchestration in Spring Boot WebFlux, ROT/Base64
obfuscation decoding, and headless-browser automation with Playwright.

The project is published so that other engineers can study these patterns.
It is **not** intended for large-scale commercial re-distribution of
third-party video content.
