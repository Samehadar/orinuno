<CardGrid stagger>
  <Card title="Kodik API client" icon="rocket">
    Full coverage of the seven Kodik REST endpoints with 70+ search filters,
    token-bucket rate limiting, and raw-response-first schema drift detection.
  </Card>
  <Card title="Video decoding" icon="puzzle">
    Brute-force ROT cipher decoder that survives shift changes, dynamic
    endpoint discovery from player JS, and retry with exponential backoff.
  </Card>
  <Card title="Download and stream" icon="seti:video">
    Playwright-backed HLS capture, parallel segment download, ffmpeg remux to
    MP4, and a streaming endpoint with Range support.
  </Card>
  <Card title="Production plumbing" icon="setting">
    Liquibase-managed MySQL schema, scheduled TTL refresh, proxy pool with
    round-robin rotation, Prometheus metrics, and optional API-key auth.
  </Card>
</CardGrid>
